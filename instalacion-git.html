Bienvenido a la sección de GIT, aquí usted aprenderá a como instalar el cliente de escritorio GIT para Windows y su configuración básica para empezar a trabajar rápidamente.

Intalación en Windows

La forma más fácil para instalar Git para Windows es accediendo a su página web oficial. Solo tienes que visitar http://git-scm.com/download/win y la descarga empezará automáticamente.

Configurando Git por primera vez

Ahora que tienes Git Instalado en ti equipo, es necesario personalizarlo mediante la herramienta llamada git config. Los cambios se mantendrán entre actualizaciones.

Lo primero que puedes realizar es establecer tu correo electrónico y nombre de usuario. Esta información es valiosa porque los "commits" de Git la envían cada vez que realizas uno, para establecerlo utiliza los siguientes comandos:

$ git config --global user.name "Nombre de Usuario"
$ git config --global user.email correo@mail.com

Para comprobar la configuración ingresada se puede usar el comando git config --list para mostrar todo lo que ha configurado.
